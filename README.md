# Simple Text Editor WinForms

Простейший текстовый редактор на Windows Forms. Открывает и сохраняет текстовые файлы в кодировке Unicode.

## Функциональность

- Открытие текстового файла из D:\Text1.txt
- Сохранение текста обратно в тот же файл
- Многострочное текстовое поле
- Обработка ошибок (файл не найден и др.)
- Русскоязычный интерфейс

## Интерфейс

- Кнопка "Открыть" — загружает текст из файла
- Кнопка "Сохранить" — записывает изменения
- Текстовое поле для редактирования

##Куда можно встроить

- Как модуль редактирования конфигурационных файлов
- В программу с настройками (блокнот для логов/заметок)
  
##Требования

- Windows
- .NET Framework / .NET Core с WinForms
- Файл D:\Text1.txt (или измените путь в коде)

##Примечание

Файл открывается и сохраняется по фиксированному пути D:\Text1.txt. Для использования в других проектах замените переменную Text1 на нужный путь.
